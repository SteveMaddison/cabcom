The MAME emulator can be used to generate a list of all the games it can run, including details such as the full title, publisher and release date.

By providing the path to the MAME Executable on your system, this list can generated and imported into the game database. The information can then be used to automatically populate your active game list with this information.

The optional Platform may be used to set the platform associated with all games imported from MAME.

Use the Save button to save the MAME configuration details. The Import button starts the import process, which may take some considerable time as most MAME versions support many thousands of individual games.
